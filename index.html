01.1 — Deducción física del sistema masa‑resorte

El sistema
masa‑resorte
sin fricción

Solo dos elementos: la masa m y el resorte k. El resorte genera una fuerza de restitución que siempre apunta hacia el equilibrio.

// Segunda Ley de Newton: \( \displaystyle\sum F = ma = m\,\frac{d^2x}{dt^2} \) // Ley de Hooke (fuerza restitutiva): \( F_r = -kx \) // Combinando: \( m\,\frac{d^2x}{dt^2} = -kx \) \( m\,\frac{d^2x}{dt^2} + kx = 0 \)
// Definición frecuencia natural: \( \omega^2 = \dfrac{k}{m} \quad\longrightarrow\quad \omega = \sqrt{\dfrac{k}{m}} \) // Ecuación canónica del MAS: \( \ddot{x} + \omega^2 x = 0 \)
Convención de signo
x > 0 → resorte estirado → Fr apunta hacia la pared (−x)
x < 0 → resorte comprimido → Fr apunta lejos de la pared (+x)
Fr siempre apunta al equilibrio: de ahí el signo negativo.
Leyenda
Fr = −kx  (fuerza del resorte)
x(t) desplazamiento desde equilibrio
Fext  (fuerza que provoca el estiramiento)

01.2 — Clasificación de la ecuación diferencial

¿Qué tipo de
ecuación es esta?

Al plantear el MAS obtenemos:

\( m\,\ddot{x} + kx = 0 \)

Esta ecuación se clasifica como Ecuación Diferencial Ordinaria Lineal Homogénea con Coeficientes Constantes, expliquemos paso a paso:

Clasificación completa
Ecuación — contiene "=" con dos miembros
Diferencial — contiene derivadas (d²x/dt²)
Ordinaria — derivadas de una sola variable (t)
Lineal — todas las derivadas con grado 1
Homogénea — cumple L[x] = 0
Coef. Constantes — m y k son números fijos
Operador Lineal
// Si L[x] = 0 → homogénea
// Si L[x] ≠ 0 → no homogénea

El orden es 2 (máxima derivada)
El grado es 1 (potencia de esa derivada)
¿Por qué importa clasificar?
La clasificación determina el método de solución. Para EDO Lineal Homogénea con Coeficientes Constantes usamos el ansatz exponencial \(x(t) = e^{rt}\), que lleva a la ecuación característica.
// Operador diferencial explícito:
\( \mathcal{L}[x] = m\,x'' + k\,x \) // \(\mathcal{L}[x] = 0\) → Ecuación homogénea ✓
01.3 — Solución: Ansatz exponencial y fórmula de Euler

Resolviendo la
ecuación del MAS

Paso 1 — Ansatz (del alemán "ansetzen" que significa "Aplicar")
Proponemos: \( x(t) = e^{rt} \)
\( x'(t) = r\,e^{rt} \)
\( x''(t) = r^2 e^{rt} \)
// Sustituyendo en \(m\ddot{x} + kx = 0\):
\( m r^2 e^{rt} + k e^{rt} = 0 \)
\( e^{rt}\!\left(r^2 + \dfrac{k}{m}\right) = 0 \)
Paso 2 — Ecuación Característica
// \(e^{rt} \neq 0\) (límite \(t\to-\infty\) físicamente imposible)
\( r^2 + \omega^2 = 0 \)
\( r^2 = -\omega^2 \)
\( r = \pm\,\omega i \)  // raíces imaginarias puras
Paso 3 — Fórmula de Euler
\( e^{i\theta} = \cos\theta + i\sin\theta \)
\( \therefore\; e^{i\omega t} = \cos(\omega t) + i\sin(\omega t) \)
// Solución general:
\( x(t) = C_1\cos(\omega t) + C_2\sin(\omega t) \)
Paso 4 — Forma Amplitud‑Fase
// Por identidad trigonométrica:
\( A\cos(\omega t+\Phi) = A\cos\Phi\cos(\omega t) - A\sin\Phi\sin(\omega t) \)
\( = C_1\cos(\omega t) + C_2\sin(\omega t) \)
\( x(t) = A\cos(\omega t + \Phi) \)
GRÁFICA — x(t) = A·cos(ωt + Φ)
t = 0.00 s

02.1 — Deducción y Diagrama de Cuerpo Libre — Sistema masa‑resorte‑amortiguador

Sistema
masa‑resorte‑
amortiguador

La fuerza de amortiguamiento es proporcional a la velocidad y se opone al movimiento. Al agregarla, aparece una segunda fuerza actuando simultáneamente sobre la masa.

// Fuerzas activas en eje x: \( F_r = -k\,x \) \( F_a = -\beta\,\dot{x} \) // Segunda Ley: ΣF = ma \( -kx - \beta\dot{x} = m\ddot{x} \) \( m\ddot{x} + \beta\dot{x} + kx = 0 \)
// Por convención: β = 2λm → ec. característica: \( r^2 + 2\lambda r + \omega^2 = 0 \) // Discriminante: \( \Delta = \lambda^2 - \omega^2 = \left(\dfrac{\beta}{2m}\right)^{\!2} - \dfrac{k}{m} \)
MAS vs Amortiguado
MAS: solo Fr → oscilación eterna
Amortiguado: Fr + Fa → energía siempre se disipa
Fa convierte energía cinética en calor, no la almacena.

CASO I
Δ < 0  |  raíces complejas conjugadas

Subamortiguado — Oscilación decreciente

\( r = \alpha \pm \beta i \)
\( \alpha = -\lambda < 0 \)
\( \beta = \sqrt{\omega^2-\lambda^2} \)
Solución de la ecuación característica
\( r^2 + 2\lambda r + \omega^2 = 0 \)
\( r = \dfrac{-2\lambda \pm \sqrt{4\lambda^2-4\omega^2}}{2} \)
\( r = -\lambda \pm \sqrt{\lambda^2-\omega^2} \)
// Si \(\lambda < \omega\) → Δ < 0 → raíces complejas:
\( r = -\lambda \pm i\sqrt{\omega^2-\lambda^2} = \alpha \pm \beta i \)
Solución general — Euler aplicado
\( x(t) = e^{\alpha t}\!\left[C_1\cos(\beta t) + C_2\sin(\beta t)\right] \)
// Forma amplitud-fase:
\( x(t) = A\,e^{-\lambda t}\cos(\beta t + \Phi) \)
// \(e^{-\lambda t}\) es la envolvente amortiguadora
GRÁFICA — x(t) subamortiguado
t = 0.00 s
ANIMACIÓN FÍSICA — Subamortiguado

CASO II
Δ > 0  |  raíces reales distintas

Sobreamortiguado — Retorno monótono lento

\( r_1 \neq r_2 < 0 \)
\( r = -\lambda \pm \sqrt{\lambda^2-\omega^2} \)
ambas reales negativas
Solución de la ecuación característica
\( r^2 + 2\lambda r + \omega^2 = 0 \)
\( r = -\lambda \pm \sqrt{\lambda^2-\omega^2} \)
// λ > ω → Δ > 0 → dos raíces reales negativas:
\( r_1 = -\lambda + \sqrt{\lambda^2-\omega^2} < 0 \)
\( r_2 = -\lambda - \sqrt{\lambda^2-\omega^2} < 0 \)
Solución general
\( x(t) = C_1 e^{r_1 t} + C_2 e^{r_2 t} \)
// \(C_1,\,C_2\) por condiciones iniciales:
\( C_1 = \dfrac{v_0 - r_2 x_0}{r_1-r_2} \)
\( C_2 = \dfrac{r_1 x_0 - v_0}{r_1-r_2} \)
// Sin oscilación: suma de dos exponenciales decrecientes
GRÁFICA — x(t) sobreamortiguado
t = 0.00 s

Amortiguamiento Crítico — Retorno óptimo

\( r_1 = r_2 = r \)
\( r = -\lambda \)
caso límite óptimo
Solución de la ecuación característica
\( r^2 + 2\lambda r + \omega^2 = 0 \)
// λ = ω → Δ = 0 → raíz doble:
\( r = -\lambda \)  (con multiplicidad 2)
// Una sola raíz → necesitamos dos soluciones independientes
Solución general — reducción de orden
// \(x_1 = e^{rt}\) → \(x_2 = t\,e^{rt}\) (por reducción de orden)
\( x(t) = (C_1 + C_2\,t)\,e^{rt} \)
// Con \(r = -\lambda\):
\( x(t) = (C_1 + C_2\,t)\,e^{-\lambda t} \)
// El factor \(t\) impide que decaiga trivialmente
